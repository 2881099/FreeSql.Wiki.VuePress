# FreeSql Docs

- http://freesql.net

## install
```bash
npm(yarn) install
```
## run
```bash
npm run docs:dev
```
## build 

```bash
npm run docs:build
```

## upgrade package
```bash
npm i vuepress
npm i @vuepress/plugin-docsearch
npm i @vuepress/plugin-pwa-popup
npm i vuepress-vite
```
